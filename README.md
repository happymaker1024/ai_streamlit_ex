# ai_streamlit_ex
openai 와 streamlit을 활용한 서비스 개발

## 기본 설치 라이브러리
```
pip install openai
pip install streamlit
```
## 실행방법
```
streamlit run 파일명.py
예시) streamlit run 2-1.dalle_app.py
```


## openai에서 이미지를 직접 리턴받을 때 필요
```
pip install Pillow
```
